# Introduction

Svelte AI Elements is a SvelteKit-first component registry for building AI-native interfaces.

## Inspired by Vercel AI Elements

The project adapts the AI Elements idea to Svelte with a focus on SvelteKit and shadcn-svelte.

## Goal

Ship AI UI faster with reusable building blocks for prompts, messages, reasoning, tools, and conversations.

## Best first steps

1. Read the installation guide.
2. Add the `prompt-input` component.
3. Continue to the component docs and guides.
