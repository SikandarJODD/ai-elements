# Confirmation

An alert-based component for managing tool execution approval workflows with request, accept, and reject states.

## Installation

<Tabs items={["CLI", "Manual"]}>
<Tab value="CLI">

### Using CLI

```bash
npx shadcn-svelte@latest add https://sv-animations.vercel.app/r/confirmation.json
```

</Tab>
<Tab value="Manual">

### Manual Installation

Copy the component source files into your project and update the example files with real usage.

</Tab>
</Tabs>

## Usage

```svelte
<script lang="ts">
  import { Confirmation } from "$lib/components/ai-elements/confirmation";
</script>

<Confirmation />
```
